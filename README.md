# NestJs QuickStart

### Modules
- Admin
- User
- Session
- Role
- Auth

### Integrations
- Typegoose
- Passportjs


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API Documentation

```bash
<APP_URL>/api
```
